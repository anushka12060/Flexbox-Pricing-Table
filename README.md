# Flexbox Pricing Table
A simple responsive pricing table built using HTML and CSS Flexbox.
## Features
- Responsive design
- Clean, modern layout
- Mobile-friendly
## Technologies Used
- HTML5
- CSS3 (Flexbox)
- Google Fonts (Sono)
## Responsive Behavior
- **Large screens**: Pricing plans appear side-by-side
- **Small screens**: Plans stack vertically using media queries
